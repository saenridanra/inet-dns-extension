Detailed Description

DNSLocalCache a simple DNS proxy implementation.

Similar to the DNSAuthServer but it only checks the cache and otherwise forwards the question to the DNSAuthServer if the question can not be answered using the cache.

Author
Andreas Rain, Distributed Systems Group, University of Konstanz
Date
March 26, 2015
