Detailed Description

Provides functionality of a mdns resolver.

New queries can be sent over the internal interface sending a cMessage to this module internally containing either message kinds:

  • MDNS_KIND_INTERNAL_QUERY : used for queries
  • MDNS_KIND_INTERNAL_PUBLISH : used to publish own services dynamically
  • MDNS_KIND_INTERNAL_REVOKE : used to take down services

Further operations may follow.
